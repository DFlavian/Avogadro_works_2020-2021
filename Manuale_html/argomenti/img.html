<img>

Descrizione

Questo tag serve a inserire uno spazio in cui mettere un immagine in una pagina HTML.

Attributi

Nome Descrizione
src Indica da dove prendere l'immagine. Senza di essa, dà un errore.
title Il testo messo in questo attributo viene mostrato quando il cursore passa sopra l'immagine.
alt Specifica un titolo alternativo quando l'immagine non viene caricata correttamente.
height Indica l'altezza dell'immagine. Può essere relativa, in %, o in pixel.
width Come l'altezza, ma si occupa della larghezza.

Formati immagini comuni

Abbreviazione Formato Estensione
APNG Animated Portable Network Graphics .apng
GIF Graphics Interchange Format .gif
ICO Microsoft Icon .ico, .cur
JPEG Joint Photographic Expert Group image .jpg, .jpeg, .jfif, .pjpeg, .pjp
PNG Portable Network Graphics .png
SVG Scalable Vector Graphics .svg
